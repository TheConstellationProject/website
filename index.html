About us

The Dark Net: It pervades culture in how "scary" it is, but in reality it's just a bunch of programmers making a bunch of bits flip. The idea of Constellation is to fill in the boots where Dark Nets (such as Tor) fail and provide a low latency, private internet within the internet and allow the user to use their own applications on it, as if nothing ever changed.

Why Constellation

Open Source Code

All code to this project can be found on GitHub under the GNU GPL v3 License, allowing you to create your own networks.

Private

To get dedicated servers on Constellation, only it's MAC address and a unique code tied to your device is shared with the network controllers.

Robust Codebase

The Constellation Project is a large project, consisting of many programmers around the globe helping to make it useful.

Cross-Compatible

Constellation Works on virtually any common OS, with APIs for programmers to port applications to exotic hardware.

Interlinked

We peer with several companies, including (but not limited to) Akamai, Cloudflare, etc. And we serve as an Internet Exchange Point.

It's Free!

Anyone can use Constellation, by simply running an installer. Now, what will you do with it?

Communicate Securely

Constellation is a network implementation, built from the ground up to be secure, reliable and fast with the help of it's GoLang codebase and high-speed network connections.

Learn More

Experiment Safely.

Programmers, rejoice! Here is a "safe space" to practice, Constellation has the ability to contain everything inside of a WAN. This allows programmers to connect their devices and do potentially destructive mistakes with less worry.

More Details

Its a Web...

Constellation functions as it's own complete network, with domains and IP addresses. Traffic is routed to the proper place, and best of all: It virtualizes a network card, so purchasing a network card to interconnect to Constellation is unneeded.
