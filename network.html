About The Network

Arachnid Network is a Software-Defined Wide Area Networking solution powered by ZeroTier, an Open Source program that allows the creation of something called intranets. To cut through all this technical jargon, many people have been to video gaming LAN parties. Arachnid does this on a world wide scale, by emulating a network card and connecting you to it's network using that. No additional hardware needed.

Arachnid Network supports Border Gateway Protocol (in other words, Arachnid has the power of an internet service provider in many regards), making Arachnid truly interconnected with the rest of the internet. Along with this, Arachnid has more than one way to get online: direct connect using the install scripts/instructions for your device (works on all major Operating Systems) or use the API to develop services that can run on Arachnid- so you can make an Arachnid powered smart home.
