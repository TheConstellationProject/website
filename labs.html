Labs

Because Constellation is it's own intranet (See About the Network for more info) and it's open source, users can utilize the code to create their own little "internets" to test potentially destructive malware, such as WannaCry, safely. Constellation also allows the creation of computer labs, spread across the globe without much work.

This Network also enables users to learn TCP/IP, Linux, and other foundational skills any IT practitioner needs to know in order to be successful. This is due to the fact that Constellation has the ability to serve as a "virtual playground" where people can experiment with code and have a website or server online without paying for domains or IP addresses.
